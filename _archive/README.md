# _archive/

This directory contains previous development iterations of the Mince application.
These folders are **read-only historical references** and are **not part of the active codebase**.

| Folder | Description |
|--------|-------------|
| `coin-collection-app-run1/` | Iteration 1 — initial Python/FastAPI backend prototype |
| `coin-collection-app-run2/` | Iteration 2 — expanded backend + basic React frontend |
| `coin-collection-app-run3/` | Iteration 3 — further backend development |
| `coin-collection-app-beh4/` | Iteration 4 — backup, reports, sharing features |
| `coin-collection-app-beh5/` | Iteration 5 — security middleware, full docs |
| `fortent-beh6/` | Iteration 6 — advanced frontend features (search, camera, PWA) |
| `backend-beh7/` | Iteration 7 — auth, scrapers, background jobs |
